# Git Tag Commit show tool

## How to use
show tag on latest tag.
```Ruby
HanedaKentarou-no-MacBook-Pro.local ^_^  ~/Documents/workspace/dev/ios/Alamofire (master)
% ruby GitTagDiff.rb                                                                                             [17:11:07]
"f129adb (tag: 3.0.0-beta.3) Added release notes to the CHANGELOG and bumped the version to 3.0.0-beta.3."
"ba280a4 Revert \"Response initializer now has internal ACL since it does not need to be public.\""
"25f35de Merge branch 'master' into tvOS"
```
